# Deep learning model for asset price bubble crash risk index

## 개요

2024년 5월 3일에 진행된 한국경영과학회 2024 춘계공동학술대회에서 발표한 연구이다. Log-periodic power Law (LPPL) 모델과 딥러닝 모델을 결합하여, LPPL 모델의 추정 결과를 보다 실용적으로 활용할 수 있게 발전시켰다.
